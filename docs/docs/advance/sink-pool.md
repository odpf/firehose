# Sink Pool

## `SINK_POOL_NUM_THREADS`

Number of sinks in the pool to process messages in parallel.

* Example value: `10`
* Type: `optional`
* Default value: `1`

## `SINK_POOL_QUEUE_POLL_TIMEOUT_MS`

Poll timeout when the worker queue is full.

* Example value: `1`
* Type: `optional`
* Default value: `1000`
